Part 1: Solidify Terminology

  • HTTP: The Hypertext Transfer Protocol (HTTP) is the protocol that is used by the World Wide Web (WWW) to request hypertext documents from one system to another. HTTPS is an encrypted form of HTTP and has become standard.
  • URL: A Universal Resource Locator (URL) is the network address of some resource
  • DNS: The Domain Name System (DNS) is a service that takes a URL and returns the corresponding IP address either from a local lookup table or from a web DNS server.
  • Query String: The query string of a URL allows the browser to pass parameters to the backend server.
  • Example: HTTP Verbs: The two most common HTTP Verbs, or methods, are the GET and the POST methods. The GET method is used to request data from the backend server via the query string while the POST method is used to pass data to the backend server via the request body.
